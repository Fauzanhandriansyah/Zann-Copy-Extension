# Zann Copy Extension 🚀

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![Browser: Chrome](https://img.shields.io/badge/Browser-Chrome-green.svg)]()

**Zann Copy Extension** is a lightweight, powerful browser extension designed to bypass text-selection and right-click restrictions on websites. If you've ever found a helpful article or snippet of code on a site that blocks you from copying it, this extension is your solution. 

## ✨ Features

*   **Re-enable Text Selection:** Unblocks websites that use CSS or JavaScript to prevent you from highlighting text.
*   **Enable Right-Click Menu:** Restores the default context menu on sites that try to disable it.
*   **Bypass Copy Restrictions:** Overrides scripts that intercept and block the `Ctrl+C` or `Cmd+C` copy commands.
*   **Lightweight & Fast:** Runs quietly in the background without slowing down your browsing experience.
*   **Plug and Play:** No complex configurations required. Install it, and it just works.
